Git History

Git is based off of an old program called "Bit Keeper." Git was developed with speed, design simplicity, the ability to handle large projects, the ability to be fully distributed, and strong support for non-linear development in mind.

Git Basics

There are multiple stages of git to move files to your remote repository. First edits should be made in the working directory, then once edits have been made in the terminal you will send the files to your local repository, once in the local repository the files will be sent to your remote repository, GitHub.

Local Repository

In this specific case our local repository is Codio. A repository holds snapshots of your files and changes that have been made to your files.
